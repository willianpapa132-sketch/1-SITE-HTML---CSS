# Projeto Front-End - HTML + CSS

Este foi meu primeiro projeto focado na construção de uma interface web utilizando apenas HTML e CSS, com o objetivo de consolidar fundamentos essenciais de desenvolvimento front-end.

## Objetivos do Projeto

- Aplicar estrutura semântica correta em HTML
- Trabalhar organização e espaçamento com divs
- Utilizar Flexbox para alinhamento e distribuição de elementos
- Implementar princípios de responsividade

## Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox

## Conceitos Aplicados

✔ Estrutura semântica (header, main, footer)
✔ Organização de layout com Flexbox
✔ Controle de espaçamento (margin, padding)
✔ Ajustes básicos para diferentes tamanhos de tela

## Resultado

O projeto simula uma página de loja virtual com:
- Área de destaque inicial
- Seção de produtos
- Layout estruturado e organizado
- Estilização consistente
